FAQ
Common questions
Everything you need to know
Is my job description data stored anywhere? +
No. Your JD is processed in memory only. ChromaDB uses in-memory storage — nothing saved to disk or any database. Each session is completely fresh and private.
How accurate is the 30-day roadmap? +
Very accurate for the specific JD you paste. The AI reads every requirement in your JD and builds the roadmap based on exactly those skills — not generic internet advice.
Does it work for any job domain? +
Yes. Works for any role — software engineering, data science, product management, design, marketing. If there's a JD, this tool can analyze it.
Why is it free? +
Currently free while in beta. We use Groq's free API tier which allows generous usage. A premium plan is coming soon for users who need unlimited analyses.
How is this different from just asking ChatGPT? +
ChatGPT gives generic answers from general knowledge. JD Analyzer uses RAG to actually read your specific JD — every answer is grounded in YOUR job description, not guesswork.
What if the analysis takes too long? +
Average time is 8–12 seconds. All 4 analyses run in parallel using Python ThreadPoolExecutor. If it takes longer, it's usually Groq API latency on the free tier — just try again.
